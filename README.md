## Only Dates

---

### Deploy link

https://only-dates.vercel.app/

---

### To install

```
git clone https://github.com/DomnitskiyOleg/only-dates.git

```

use yarn ?

```
yarn
```

use npm ?

```
npm install
```

### To start

```
npm start
```

or

```
yarn start
```
